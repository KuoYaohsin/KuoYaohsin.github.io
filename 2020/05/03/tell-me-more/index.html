先在cmd命令中输入E： 如果你的文件在其他盘就输入其他的盘。现在进入E盘了。再输入cd my software\photoshop cs6\Photoshop CS6。注意，cd后有一个空格，文件夹的大小写也要注意。这个步骤也可以分开打，如cd my software回车，cd photoshop cs6回车,cd Photoshop CS6回车。
最后输入start Photoshop.exe回车文件就打开了。
　　Windows 命令提示符（cmd.exe）是 Windows NT 下的一个用于运行 Windows 控制面板程序或某些 DOS 程序的shell程序；或在 Windows CE 下只用于运行控制面板程序的外壳程序。
